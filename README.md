# 🌦️ Weather Forecast Web App

A responsive web application that provides real-time weather updates and multi-day forecasts using a public weather API. Built with HTML, CSS, and JavaScript, the app features dynamic UI rendering, city-based search functionality, and seamless API integration to deliver accurate and user-friendly weather information.

## 🚀 Features
- 🌍 Search weather by city name  
- 🌡️ Real-time temperature, humidity, and conditions  
- 📅 5-day weather forecast  
- 📱 Responsive design (mobile + desktop)  

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  
- Weather API
